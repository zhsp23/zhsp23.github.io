操作符表

一元操作符

操作符 函数
+a a.unaryPlus()
-a a.unaryMinus()
!a a.not()
a++ a.inc()
a– a.dec()

二元操作符

操作符 函数
a + b a.plus(b)
a - b a.minus(b)
a * b a.times(b)
a / b a.div(b)
a % b a.mod(b)
a in b a.contains(b)
a..b a.rangTo(b)
a !in b !a.contains(b)
a += b a.plusAssign(b)
a -= b a.minusAssign(b)
a *= b a.timesAssign(b)
a /= b a.divAssign(b)
a %= n a.modAssign(b)

数组操作符

操作符 函数
a[i] a.get(i)
a[i,j] a.get(i,j)
a[i_1,…,i_n] a.get(i_1,…,i_n)
a[i] = b a.set(i, b)
a[i,j] = b a.set(i, j, b)
a[i_1,…,i_n] = b a.set(i_1, …, i_n, b)

等于操作符

操作符 函数
a == b a?.equals(b)?:b === null
a != b !(a?.equals(b)?:b === null)

相等操作符有一点不同，为了打倒正确合适的相等检查做了更复杂的转换，因为要得到一个确切的函数机构比较，
